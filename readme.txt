This project is to run entire system including database, back-end, and font-end in one command using docker compose.

>>>Precondition:
Docker is running in machine which will host the system

>>>Steps:
- clone this project to your machine which will host the system
- add execution mode: chmod 777 run-my-system.sh
- start system: ./run-my-system.sh

>>>Using system:
- Using browser to access system, for example: localhost:8080
- Enter a user name then click on 'Add User' button. The new added user should be shown in following list.
